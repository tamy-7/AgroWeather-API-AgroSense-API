# AgroWeather-API-AgroSense-API
AgroSense es una API REST que unifica múltiples fuentes meteorológicas para el sector agrícola. Permite consultar pronósticos precisos por ubicación y genera recomendaciones inteligentes de favorabilidad para actividades clave como siembra, riego o cosecha, optimizando la toma de decisiones mediante la consolidación de datos climáticos.
